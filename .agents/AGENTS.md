# Инструкции для ИИ-помощника (AI Agent / Copilot)

Привет! Ты помогаешь разработчику или студенту работать с проектом **GeminiGen Studio**.

## 🚀 Как развернуть и запустить проект локально
Если пользователь просит тебя развернуть или запустить проект, выполни следующие шаги:

1. **Установи зависимости**:
   ```bash
   npm install
   ```
2. **Запусти сервер разработки**:
   ```bash
   npm run dev
   ```
3. **Выведи ссылку**: После запуска Vite найди адрес локального хоста в логах (обычно `http://localhost:5173` или `http://localhost:5174`) и обязательно отправь его пользователю в итоговом сообщении.

## 🛠️ Архитектура и стек
* **Стек**: Vite + React + TypeScript.
* **CORS Прокси**: Локальный прокси-сервер настроен в `vite.config.ts` для перенаправления запросов к API, чтобы обойти CORS в браузере:
  * `/api/...` -> `https://api.geminigen.ai/...`
  * `/api-snapgen/...` -> `https://api.snapgen.ai/...`
* **API Ключ**: Передается в заголовке `x-api-key`. Пользователь вводит его на странице, и он хранится в `localStorage` (безопасно, секретные ключи не коммитятся в репозиторий).
