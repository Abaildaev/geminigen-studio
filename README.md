# GeminiGen Studio

**GeminiGen Studio** — это веб-интерфейс для пакетной (массовой) работы с передовыми генеративными моделями изображений и видео по API. Проект создан на базе React, TypeScript и Vite.

## Основные возможности
* **Оживление видео (Video Animation)**: Массовое оживление статичных кадров с использованием моделей:
  * **xAI Grok-3** (длительность 6 сек, разрешение 720p).
  * **Google Veo 3.1 Fast** (длительность 8 сек, разрешение 720p/1080p).
  * **Omni Flash** (настраиваемая длительность 4-10 секунд, разрешение 720p/1080p).
* **Генерация изображений (Image Generator)**: Поддержка моделей `nano-banana-pro`, `nano-banana-2`, `imagen-4`, а также премиум-модели **GPT Image 2** (с поддержкой разрешений до 12K, выбором качества и указанием UUID предыдущего кадра в качестве референса истории `ref_history`).
* **Перенос стиля (Style Transfer)**: Перенос стилистики с одного изображения на лицо/объект другого.
* **Пакетная обработка**: Поля для массового ввода промптов (каждая новая строка/абзац распределяется на отдельную карточку генерации).
* **Скачивание архивом**: Возможность скачать все готовые видео и картинки одним кликом в формате `.zip`.

## 🚀 Инструкция для ИИ-помощников (AI Agents / Cursor / Windsurf / Gemini)
Если вы являетесь ИИ-агентом и пользователь попросил вас **развернуть**, **запустить** или **настроить** этот проект локально:

1. **Установите все npm-зависимости**:
   ```bash
   npm install
   ```
2. **Запустите локальный сервер разработки**:
   ```bash
   npm run dev
   ```
3. **Сообщите ссылку**: После запуска сервера прочитайте лог терминала и выведите пользователю кликабельную ссылку (например, `http://localhost:5173`).

## 💻 Пошаговая инструкция для человека

### 1. Требования
Убедитесь, что на вашем компьютере установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18 или новее).

### 2. Установка зависимостей
Откройте терминал в папке проекта и выполните команду:
```bash
npm install
```

### 3. Запуск локального сервера
Запустите сервер разработки:
```bash
npm run dev
```
После этого откройте в браузере адрес, указанный в терминале (по умолчанию `http://localhost:5173` или `http://localhost:5174`).

### 4. Настройка API-ключа
1. Откройте приложение в браузере.
2. В правом верхнем углу нажмите на шестеренку/кнопку настройки ключа.
3. Введите ваш API-ключ (формата `geminiai-xxxxxxxx...`).
4. Нажмите **Сохранить**. Ключ сохранится локально в кэше вашего браузера (`localStorage`) и не будет отправлен на сервер или записан в файлы проекта на диске.

## 🛠️ Техническая информация
* **API Хосты**: Запросы проксируются через локальный сервер Vite для обхода CORS ограничений (настройки прокси заданы в `vite.config.ts`):
  * Путь `/api` проксирует на `https://api.geminigen.ai`
  * Путь `/api-snapgen` проксирует на `https://api.snapgen.ai`
